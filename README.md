# theosakram.dev

Personal portfolio of **Theophany Sakra Muhammad** — Senior Frontend Engineer.

## Stack

- **Next.js 16** — App Router
- **React 19** — with server components
- **Tailwind CSS 4** — utility-first styling
- **Motion** — scroll-driven and mount animations
- **Lenis** — inertial smooth scroll
- **next-themes** — dark/light mode

## Dev

```bash
bun dev
```

## Structure

```
src/
├── app/              # Root layout, global styles
└── components/
    ├── sections/     # Page sections (Hero, Work, Career, Skills, About, Contact)
    └── ui/           # Shared primitives (ThemeToggle, SmoothScroll, DotPattern)
```
